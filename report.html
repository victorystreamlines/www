SSH Setup and VPS Connection Guide

Clear and easy steps to set up connection to your Hostinger server

1
Login to VPS via Terminal
LocalHost Terminal (Your Computer)

Why this step?

The first step to connect to the VPS is to log in using SSH from your computer. This allows you to have full remote control of the server.

Required Command
ssh root@YOUR_VPS_IP

Note: Replace YOUR_VPS_IP with your server's IP address (Example: 185.123.45.67)

Expected Success Response

ECDSA key fingerprint is SHA256:xxxxxxxxxxxxx.
Are you sure you want to continue connecting (yes/no)?

Type: yes then press Enter

Possible Error: Connection Refused
ssh: connect to host 185.123.45.67 port 22: Connection refused

Install PHP SSH2 Extension on LocalHost
LocalHost Terminal (Laragon/XAMPP)

Why this step?

PHP needs a special extension to connect via SSH. Without this extension, an error will appear in index.php

To check if it's installed:

Required Command
